# Project Screenshots

This folder contains screenshots of completed projects to display in the portfolio section.

## Adding Screenshots

### For VSR Snow Removal

1. Visit https://vsrsnow.com
2. Take a full-page screenshot (recommended tools):
   - **Chrome/Edge**: Press F12 → Ctrl/Cmd + Shift + P → Type "Capture full size screenshot"
   - **Firefox**: Right-click → "Take Screenshot" → "Save full page"
   - **Online tool**: https://www.screenshotmachine.com or https://screenshot.guru
3. Save the screenshot as `vsr-screenshot.png` in this folder
4. Recommended dimensions: 1200x800 pixels or similar 3:2 aspect ratio

### For Future Projects

Save screenshots using this naming convention:
- `projectname-screenshot.png` (e.g., `citylyfe-screenshot.png`)
- Use lowercase and hyphens
- PNG or JPG format
- Recommended size: 1200x800 pixels

### Quick Screenshot Tips

1. **Crop out browser chrome**: Only show the website content
2. **Full-width**: Capture the full width of the site
3. **Above the fold**: Show the hero section and main content
4. **Optimize file size**: Compress images to under 500KB
   - Use https://tinypng.com or https://squoosh.app
